# Validate the deployment
{: .no_toc }

This phase confirms that your NBS 7 environment is functioning correctly before you go live. Run smoke tests, inspect monitoring and admin interfaces, and complete the cutover checklist. For many jurisdictions, this cutover moves from a prepared migration environment to production rather than applying changes directly on the primary NBS 6 production server.
