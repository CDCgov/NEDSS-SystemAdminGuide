# Maintain NBS 7

This section covers operational tasks to keep your NBS 7 environment healthy after go-live. It is intended for system administrators who are responsible for the infrastructure that runs a production NBS 7 deployment.

Maintenance tasks fall into two broad categories. **Upgrade procedures** keep your environment current such as component upgrades, and **configuration procedures** to address runtime settings that affect system behavior, such as SQL Server trace flags that improve diagnostic output for NBS application errors.

Articles in this section are added as new procedures are documented. If a procedure you need is not yet covered here, reach out to [nbs@cdc.gov](mailto:nbs@cdc.gov).
